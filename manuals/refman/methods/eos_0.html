eos//0¶

Description¶

eos

This non-terminal matches the end-of-input. It is implemented by checking that the implicit difference list unifies with []-[].

Modes and number of proofs¶

eos - zero_or_one

Errors¶

(none)

Examples¶

abc --> a, b, c, eos.

See also

``endif/0``
===========

Description
-----------

::

   endif

Ends conditional compilation for the matching :ref:`directives_if_1`
directive.

Template and modes
------------------

::

   endif

Examples
--------

::

   :- if(date::today(_,5,25)).

       :- initialization(write('Happy Towel Day!\n')).

   :- endif.

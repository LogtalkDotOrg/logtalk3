gensym¶

Predicates for generating unique atoms. Object protocol based on the gensym module of SWI-Prolog.

Author: Paulo Moura
Version: 1:0:1
Date: 2011-11-09
Compilation flags:
static, context_switching_calls
Dependencies:
(none)
Remarks:
(none)

(none)

Public predicates¶

reset_gensym/0¶

Resets the generator counter for all bases.

Compilation flags:
static, synchronized
Mode and number of proofs:
reset_gensym - one

reset_gensym/1¶

Resets the generator counter for a given base.

Compilation flags:
static, synchronized
Template:
reset_gensym(Base)
Mode and number of proofs:
reset_gensym(+atom) - one

gensym/2¶

Returns a new unique atom with a given base (prefix).

Compilation flags:
static, synchronized
Template:
gensym(Base,Unique)
Mode and number of proofs:
gensym(+atom,-atom) - one

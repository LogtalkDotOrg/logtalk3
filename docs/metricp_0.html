Remarks

Usage
This protocol should be implemented by any metric added to the system.
Item
A term identifying an item of interest to be measured. This is normally an entity.
Score
Score can be any type of term necessary to explain the nature of the item and its relationship to the metric in question. One shouldn't constrain onself to numbers in the implementation.
